# Atom
Atom은 더 이상 쪼갤 수 없는 단위에 해당하는 요소에 해당한다. 
혹은 애니메이션, 폰트, 색상 등과 같이 요소가 아닌 속성도 Atom에 해당된다. 
가장 작은 단위인 만큼 가장 많이 재사용된다고 볼 수 있다.
(https://kciter.so/posts/effective-atomic-design)
